¿Quien inventó el sistema de control de versión Git y por que?
Git (pronunciado «guit»/gɪt​) es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

¿Aquien pertenece actualmente Github y por que?
Actualmente pertenece a Microsoft quien lo ha comprado GitHub por 7.500 millones de dólares.

¿Hay otra forma que no sea la terminal para trabajar con Github?
GitHub Desktop es un cliente Git GUI gratuito y de código abierto

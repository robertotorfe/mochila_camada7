
a. ¿Quién inventó el sistema de control de versión Git y por qué?
    Fue creado por inus Torvalds,  pensando la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
b. ¿A quién pertenece actualmente Github y por qué?
    Pertenece a Microsoft, porque fue adquirida en 2018 por un importe de 7500.
c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
    Se pueden utilizar ortos programas de manera grafica como por ejemplo TortoiseGit.

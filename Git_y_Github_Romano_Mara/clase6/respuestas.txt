a. ¿Quién inventó el sistema de control de versión Git y por qué?
    Linus Torvalds fue quien inventó el sistema de control de version Git, con el fin de mejorar la eficiencia, la confiabilidad y 
compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente. 

b. ¿A quién pertenece actualmente Github y por qué?
    actualmente le pertenece a Microsoft, la compra fue justificada para acelerar el uso empresarial de GitHub y llevar las 
herramientas y servicios para desarrolladores de Microsoft a nuevas audiencias.
